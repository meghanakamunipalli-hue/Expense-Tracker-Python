A command-line Expense Tracker application built using Python and JSON file handling.

FEATURES:

1. Add Expenses
2. View Expenses
3. Calculate Total Expenses
4. Delete Expenses
5. Update Expenses
6. Search Expenses
7. Save Expenses using JSON
8. Load Expenses automatically
9. Input validation

TECHNOLOGIES USED:

1. Python
2. JSON
3. File Handling
4. Exception Handling

Concepts Used:

1. Functions
2. Lists
3. Dictionaries
4. CRUD Operations
5. JSON Handling
6. Input validation
